# WeatherApp
## React &amp; TS Simple Weather App

This App was developed using the following technologies:

* TypeScript
* React
* Redux Tools Kit
* Axios
* Ant Design
* Jest
* Enzyme

# Running the App

to setup the app after clone run 
```sh
npm install
```
to run the app run 
```sh
npm start
```

to run the tests run
```sh
npm run test
```
