# WeatherApp
## React &amp; TS Simple Weather App

This App was developed using the following technologies:

* TypeScript
* React
* Redux Tools Kit
* Axios
* Ant Design
* Jest
* Enzyme

# Running the App

to run the app use 
```sh
npm start
```

to run the tests use
```sh
npm run test
```
